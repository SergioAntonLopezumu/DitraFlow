# DitraFlow

Plataforma Web inteligente para el análisis de madurez digital en PYMES y generación de hojas de ruta de transformación digital.

---

## Descripción

DitraFlow es una aplicación Web desarrollada como Trabajo Fin de Grado (TFG) en la Universidad de Murcia cuyo objetivo es reducir la incertidumbre en los procesos de digitalización de pequeñas y medianas empresas (PYMES).

La plataforma permite:

- Evaluar el nivel de madurez digital de una PYME.
- Analizar factores críticos de éxito y barreras de transformación digital.
- Generar recomendaciones personalizadas.
- Estimar probabilidades de éxito mediante modelos de Machine Learning explicables.
- Crear hojas de ruta adaptadas a cada empresa.
- Interactuar con un asistente inteligente orientado a digitalización empresarial.
